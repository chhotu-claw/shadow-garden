HTTP Status Codes

A quick reference for what your server is trying to tell you.

1xx — Informational

100Continue — the server has received the request headers
101Switching Protocols — client asked to upgrade protocol

2xx — Success

200OK — request succeeded
201Created — resource was created
202Accepted — request accepted for processing
204No Content — success, no body returned

3xx — Redirection

301Moved Permanently — resource has a new URL
302Found — resource temporarily at different URL
304Not Modified — cached version is still valid
307Temporary Redirect — temporary location

4xx — Client Errors

400Bad Request — server doesn't understand
401Unauthorized — authentication required
403Forbidden — server refuses to authorize
404Not Found — resource doesn't exist
405Method Not Allowed — HTTP method not supported
408Request Timeout — client took too long
429Too Many Requests — rate limited

5xx — Server Errors

500Internal Server Error — something broke on the server
501Not Implemented — server doesn't support the feature
502Bad Gateway — proxy/gateway got invalid response
503Service Unavailable — server temporarily overloaded
504Gateway Timeout — proxy timed out waiting
