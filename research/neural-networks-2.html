How Neural Networks Work

March 2, 2026

Neural networks are the engine behind modern AI. They're not actually brains — but they're loosely inspired by them. Here's how they work.

The Basic Idea

A neural network is a function approximator. Input goes in, output comes out. In between are layers of "neurons" — mathematical operations that transform the data.

For example, to classify an image as "cat" or "dog":

  • Input: pixels (values 0-255)
  • Hidden layers: transform the data, extract features
  • Output: probability of cat vs dog

Neurons: The Basic Unit

A single neuron does:

output = activation(sum(inputs × weights) + bias)

The activation function introduces nonlinearity — allowing the network to learn complex patterns. Common activations:

  • ReLU — max(0, x), most common
  • Sigmoid — squashes to 0-1
  • Tanh — squashes to -1 to 1

Layers: Building the Network

Neurons are organized into layers:

  • Input layer — receives raw data
  • Hidden layers — extract features, learn patterns
  • Output layer — produces final prediction

"Deep" learning just means many hidden layers.

Training: Learning from Data

Networks learn through backpropagation:

  1. Forward pass: compute output
  2. Compute loss: how wrong was the prediction?
  3. Backward pass: compute gradients
  4. Update weights: gradient descent

This is repeated millions of times until the network learns to predict correctly.

What They Learn

Each layer learns increasingly abstract features:

  • Layer 1: edges, colors
  • Layer 2: shapes, textures
  • Layer 3: object parts
  • Layer 4: complete objects

Types of Networks

  • MLP — classic fully-connected
  • CNN — convolutional, great for images
  • RNN/LSTM — sequential data, text, time series
  • Transformer — attention mechanism, language models

What Remains

Neural networks are function approximators — they learn patterns from data. They're not intelligent in the human sense. They don't understand; they memorize statistical correlations.

But they work. Better than anything else we've tried for many tasks. That's the mystery — why do they work so well?
