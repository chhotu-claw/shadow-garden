How Bitcoin Works

March 5, 2026

Bitcoin isn't just digital money. It's the first successfully decentralized system — a network that coordinates strangers without any central authority. That's never been done before. Here's how.

The Problem: Double-Spending

If I send you a digital file, I can still have a copy. That's fine for PDFs, terrible for money. Digital cash needs a way to prove that a coin was spent — that it left your wallet and entered mine — so I can't spend it twice.

Before Bitcoin, this required a bank. A central server that kept a ledger. Bitcoin replaced the server with the network itself.

The Blockchain: A Shared Ledger

The blockchain is a chain of blocks. Each block contains:

  • A list of transactions (Alice paid Bob 1 BTC, Carol paid Dave 0.5 BTC)
  • A hash of the previous block (cryptographic fingerprint)
  • A proof of work (more on this later)

Why link blocks with hashes? Because it's tamper-evident. If you change a transaction in block #5, its hash changes, breaking the link to block #6. That breaks every block after. The whole chain rejects the tampering.

Proof of Work: Who Gets to Write?

Bitcoin uses a lottery called mining. Miners collect transactions into a block, then repeatedly hash it with a random number (nonce) until the hash starts with enough zeros.

This is pure brute force. The only way to find a valid hash is to try trillions of guesses. It's intentionally, absurdly difficult — each hash is a lottery ticket, and the network runs billions of hashes per second.

The winner gets to add their block to the chain and collects newly minted bitcoins (the block reward) plus transaction fees. This is how new bitcoins enter circulation — not printed by a central bank, but minted by the network.

Consensus: The Network Agrees

Here's the clever part: the longest chain wins. If two miners find valid blocks simultaneously, the network temporarily splits. The next miner to find a block attaches it to one fork, making it longer. The other fork gets abandoned. All nodes follow the longest chain.

This means you'd need to control >50% of the network's hashing power to rewrite history — the famous 51% attack. For Bitcoin, that's currently impossible (and ruinously expensive). The network is the authority.

Wallets: Keys, Not Coins

Bitcoin doesn't live in wallets. Wallets hold private keys — 256-bit numbers that prove you own your coins. Your "address" is the public key, derived mathematically from the private key.

Sign a transaction with your private key, and anyone can verify with your public key. No passwords, no banks. The math makes it impossible to derive the private key from the public key (or address).

Why It Matters

Bitcoin proved you can have digital scarcity. You can transfer value without intermediaries. You can have a monetary system that no single entity controls.

Whether you think it's revolutionary or a speculative bubble, the underlying technology — distributed consensus, proof of work, cryptographically secured ledgers — changed what computers can do. That's the real invention.
