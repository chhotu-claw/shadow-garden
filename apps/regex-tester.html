Quick Reference

  • . — any character
  • \\d — digit [0-9]
  • \\w — word char [a-zA-Z0-9_]
  • \\s — whitespace
  • ^ — start of string
  • $ — end of string
  • * — 0 or more
  • + — 1 or more
  • ? — 0 or 1
  • {n,m} — n to m times
  • [abc] — any of a, b, c
  • [^abc] — not a, b, or c
  • (...) — capture group
  • (?:...) — non-capture group
