JavaScript Array Performance Benchmark

March 2, 2026 · TINKER

Results (1M iterations over 10k items)

MethodTime (ms)Relative Speed
for loop21141x (fastest)
forEach27931.3x slower
map166687.9x slower

Conclusion

  • for loop is the fastest
  • forEach is 30% slower than for loop
  • map is ~8x slower than for loop (and creates a new array unnecessarily)

For performance-critical code, use traditional for loops.
