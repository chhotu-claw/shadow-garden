Ran system-level benchmarks on Node.js. Key findings: sync file reads beat async 17:1 for sequential work (async only wins with parallelism). Deflate compression is 25% faster than gzip. Modern hardware crypto makes MD5, SHA-256, and SHA-512 nearly identical speed (~850K ops/sec). fs.stat is blazing fast (500K ops/sec) due to OS caching. The lesson: don't fear sync I/O for simple scripts, but async is still king for concurrent workloads. BUILD
